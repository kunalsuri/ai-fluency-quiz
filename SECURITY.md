# Security Policy

## Supported Versions

This project is currently maintained on the `main` branch only.

| Version | Supported |
| :--- | :---: |
| `main` | ✅ |
| older snapshots/forks | ❌ |

## Reporting a Vulnerability

Please report security issues privately via GitHub Security Advisories:

1. Open the repository Security tab.
2. Click **Report a vulnerability**.
3. Provide reproduction steps, impact, and any proof-of-concept details.

Direct link:
<https://github.com/kunalsuri/ai-fluency-quiz/security/advisories/new>

Please do not open public issues for undisclosed security vulnerabilities.

## What to Include in a Report

- A clear description of the issue and affected files/features.
- Reproducible steps (including browser/OS if relevant).
- Expected behavior vs. actual behavior.
- Potential impact and suggested remediation (if known).

## Response Process

- We will acknowledge valid reports as quickly as possible.
- We will investigate, triage severity, and provide status updates.
- Fixes will be released on `main`, with credit if you want to be acknowledged.
